# UKbd 示例

## 基础使用

```vue
<template>
  <p>按下 <u-kbd>Ctrl + S</u-kbd> 保存</p>
  <p>按下 <u-kbd>Esc</u-kbd> 关闭</p>
</template>
```

## 组合快捷键

```vue
<template>
  <p>
    <u-kbd>⌘</u-kbd> + <u-kbd>K</u-kbd> 打开命令面板
  </p>
  <p>
    <u-kbd>Ctrl</u-kbd> + <u-kbd>Shift</u-kbd> + <u-kbd>I</u-kbd>
  </p>
</template>
```
