# 优化前端 Header 样式设计

> 状态: 已执行

## 目标

重新设计应用顶部 Header 的视觉样式，使其更精致、现代，与 Sidebar 的设计语言保持一致。当前 Header 过于简朴（纯 bg-card/90 背景、h-11 较矮、元素缺乏层次感），需要提升视觉品质。

## 内容

### 1. 增强 Header 整体视觉层次

- 将 Header 高度从 `h-11`（44px）调整为 `h-12`（48px），与 Sidebar logo 区域（h-14）更协调
- 优化背景样式：使用更细腻的背景效果，保持 `backdrop-blur` 但增强层次感
- 底部边框增加微妙的阴影效果，增强与内容区域的分隔感

### 2. 优化面包屑导航区域

- 为当前页面标题增加更醒目的样式（增大字重或尺寸）
- 在面包屑前添加一个对应当前路由的图标，与 Sidebar 导航项呼应
- 优化面包屑分隔符的视觉效果

### 3. 优化右侧工具栏

- 统一图标按钮的 hover 效果，增加更精致的交互反馈
- 在工具按钮组和用户头像之间添加细微的分隔线
- 优化用户头像区域：显示用户名文本（桌面端），增强可识别性

### 4. 验证

- 运行 `cd web && bun run lint` 检查代码
- 运行 `cd web && bun run build` 确认构建通过
- 确保深色/浅色主题下均表现良好
- 确保响应式布局在移动端正常工作

## 影响范围

- `web/src/components/layout/header.tsx` — 完整重写 Header 组件样式
