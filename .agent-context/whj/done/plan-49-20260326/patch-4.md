# 优化构建进度 UI

## 补丁内容

用户反馈构建进度部分较丑，因此重新设计构建进度（进度条/步骤条）组件，采用现代化和响应式 UI（UI/UX Pro Max 推荐方案）：
1. 进度连线改为占满剩余宽度的弹性伸缩（`flex-1`），实现连贯线段的效果；当前状态到未完成状态过渡线设计更清晰。
2. 增加当前激活步骤（active）图标周围的动态波纹（Ping）高亮动效，提示正在进行。
3. 将阶段标签放在节点下方并绝对定位，保持圆圈垂直居中连线。
4. 优化各状态的配色和元素层级关系，带来更精美的视觉反馈。
