**请求体**: （无则整节省略）

`body` 为对象时展开真实字段：

| 字段名    | 类型                   | 必填 | 说明         |
| --------- | ---------------------- | ---- | ------------ |
| tableName | string                 | 是   | 主表表名     |
| data      | Record<string, object> | 是   | 主表字段与值 |

整段 `body` 不为对象（标量 / 数组等）时，字段名用 `_(body)_`：

| 字段名     | 类型     | 必填 | 说明               |
| ---------- | -------- | ---- | ------------------ |
| _(body)_   | number[] | 是   | 待提交记录 ID 列表 |

**响应体**（即信封里的 `data`）:

| ------ | ------ | ------ |
| id     | string | 记录 ID |
| name   | string | 名称   |

`data` 不为对象（标量 / 数组等）时，字段名用 `_(data)_`：

| 字段名    | 类型    | 说明     |
| --------- | ------- | -------- |
| _(data)_  | boolean | 是否成功 |
