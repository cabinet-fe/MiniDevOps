## 1. 阶段目标

四类 AI CLI **并行**交付；智能体定义/触发/运行；开放 Agent Skills；PAT 下载。

---

## 2. 范围说明

### 2.1 本阶段交付（摘要）

| 类别 | 内容 |
| --- | --- |
| CLI | Claude Code、OpenCode、Reasonix、Codex：检测/安装/升级/卸载、多源回退、运行时注入配置 |
| 智能体 | AiAgent、AgentTrigger（手动/API/定时/构建事件）；AgentRun 独立异步；上下文 = 提示词 + 仓库 |
| 构建事件 | 默认 `artifact_ready`（归档成功、制品可用）；BuildJob 可覆盖为 `distribution_finished` |
| Skills | ZIP 上传、SKILL.md 校验、public/private、覆盖更新、工作区注入 |
| PAT | 哈希存储、仅显示一次、可过期/吊销；固定 scope：`skills:read`、`agents:run` |
| 文档生成 | 接通 P3：`docs/generate` → AgentRun → 只写 draft |
| 前端 | CLI 管理、智能体、触发器、运行日志 WS、Skills、PAT 管理 |
| 测试 | 四 CLI 安装与非交互运行；构建事件不改 BuildRun 成功态；PAT 鉴权；私有 Skill 隔离 |

### 2.2 明确不做

- [ ] ~~自动注入需求/文档/历史会话为上下文~~（首期仅提示词 + 仓库）
- [ ] ~~远程 Runner / OS 沙箱隔离~~
- [ ] ~~Skill 版本历史~~（覆盖更新）
- [ ] ~~非功能容量调优、多节点调度~~

### 2.3 关键依赖

- 构建事件触发 → 依赖 P1 BuildRun 生命周期事件
- 文档生成接通 → 依赖 P3 草稿模型；本阶段实现 AgentRun → 只写 draft
- **禁止**在构建流水线内同步跑 Agent

## 3. Gate 条件（退出准则）

来源：[ROADMAP.md §6.4](../ROADMAP.md#64-退出-gate)

1. 四种 CLI 均可完成检测与至少一条参考运行路径（健康或可观测失败）。
2. 手动/API/Cron/构建事件均可创建独立 AgentRun；失败不影响已成功 BuildRun。
3. 合法 Skill ZIP 可上传；缺 SKILL.md 拒绝；更新后下载为新包。
4. PAT：无效 401；scope 外端点 403；私钥仅创建时回显一次。
5. 文档生成写入 draft，需人工发布。

另须满足 [ROADMAP.md §9 通用质量门禁](../ROADMAP.md#9-每阶段通用质量门禁)。

## 4. 详细开发任务清单

### 4.1 契约与数据模型

- [ ] OpenAPI：AI CLIs、Agents、Triggers、Runs、Skills、PAT（Tokens）
- [ ] OpenAPI：接通 `projects/.../docs/generate` 真实行为（相对 P3 stub）
- [ ] 再生 3.1 投影
- [ ] migration：`CliRuntimeDefinition` / `CliInstallSource` / `CliInstallJob`（或等价命名）
- [ ] migration：`AiAgent`、`AgentTrigger`、`AgentRun`
- [ ] migration：`SkillPackage`（元数据 + StorageObject 引用）
- [ ] migration：`PersonalAccessToken`（哈希、scope、过期、吊销）
- [ ] 种子：`ai.*` 菜单与权限；四 CLI 定义与默认安装源
- [ ] BuildJob 配置字段：构建事件触发 Agent 绑定 + 事件类型（默认 `artifact_ready`，可覆盖 `distribution_finished`）

### 4.2 后端 — AI CLI（`internal/ai`）

- [ ] 四 CLI 并行实现：Claude Code、OpenCode、Reasonix、Codex
- [ ] 列表：名称、key、二进制、版本、安装状态、路径、健康
- [ ] detect / install / upgrade / uninstall；多源优先级回退（策略对齐运维工具链）
- [ ] 运行时注入配置（API Base、环境变量模板、默认参数）；不强制覆盖 CLI 自身登录态文件
- [ ] CliInstallJob：异步 + 重启协议（queued 恢复；running→interrupted）
- [ ] 同 UID 执行风险审计与用户可见说明

### 4.3 后端 — 智能体与运行

- [ ] AiAgent CRUD：name、description、enabled、cli_key、system_prompt、skill_ids、repository_id、timeout
- [ ] AgentTrigger：手动、API、定时（Cron）、构建事件
- [ ] 手动触发：`POST /ai/agents/:id/runs`（需 `ai.agents:execute` 等）
- [ ] API 触发：JWT 或 PAT（scope `agents:run`）
- [ ] 定时触发：IANA 时区；禁重叠；停机错过跳过（对齐 Cron 协议）
- [ ] 构建事件：默认 `artifact_ready`；Job 可覆盖 `distribution_finished`；**异步**创建 AgentRun
- [ ] AgentRun：独立状态机；上下文 = 系统提示词 + 选定仓库；显式绑定 Skill 注入工作区
- [ ] Agent 失败**不**修改 BuildRun.status / 不改已成功构建态
- [ ] 日志文件 + WS `/ws/ai/runs/:id/logs`；终态通知
- [ ] 重启：queued 恢复；running → interrupted；重试建议新 Run
- [ ] **禁止** Pipeline 内嵌同步 agent 阶段

### 4.4 后端 — Skills

- [ ] ZIP 上传经 StorageService（默认 50MB）；校验存在 `SKILL.md`，否则 422
- [ ] 防 Zip Slip / 解压炸弹
- [ ] 可见性：`public`（有 Skills 查看权限可见）/ `private`（仅创建者；首期不做对象 ACL）
- [ ] 覆盖更新：不保留历史版本；Run 快照保存 package digest
- [ ] 下载 ZIP：JWT 或 PAT scope `skills:read`
- [ ] Agent 运行时按绑定 skill_ids 注入工作区

### 4.5 后端 — PAT

- [ ] 创建：生成明文仅回显一次；存储哈希
- [ ] scope ⊆ {`skills:read`, `agents:run`}
- [ ] 过期、吊销；无效 token → 401
- [ ] scope 外端点 → 403
- [ ] 与 JWT 分流校验（`Authorization: Bearer`）

### 4.6 后端 — 接通 P3 文档生成

- [ ] `POST .../docs/generate`：创建 AgentRun；成功后**只写** `draft_content`（及 draft 元数据）
- [ ] 可选写入 `draft_source_run_id`
- [ ] 不自动 publish；发布仍走 P3 确认 + `expected_version`

### 4.7 前端（`web-v2/`）

- [ ] AI CLI 管理页：检测/安装/升级/卸载、源管理、健康状态
- [ ] 智能体 CRUD；触发器配置（含构建事件类型选择）
- [ ] 运行列表/详情；日志 WS
- [ ] Skills：上传、可见性、覆盖更新、下载
- [ ] PAT 管理：创建（一次回显）、列表元数据、吊销；明文不回读
- [ ] 项目管理文档页：生成入口接通；结果进入草稿；引导人工发布
- [ ] BuildJob 配置：可选绑定 Agent 与事件类型
- [ ] 菜单由服务端下发；风险提示（同 UID）可见

### 4.8 测试

- [ ] 四 CLI：检测 + 至少一条参考运行路径（成功或可观测失败）
- [ ] 手动 / API / Cron / 构建事件均可产生独立 AgentRun
- [ ] Agent 失败后已 success 的 BuildRun 状态不变
- [ ] 缺 SKILL.md 的 ZIP 拒绝；合法包可上传下载；覆盖后下载为新内容
- [ ] 私有 Skill：非创建者不可见/不可用
- [ ] PAT：无效 401；scope 外 403；明文仅创建响应出现一次
- [ ] 文档生成只写 draft；需 publish 才进 published
- [ ] 三库合同测试（Agent/Skill/PAT 关键表）

### 4.9 文档

- [ ] 声明：无 OS/容器沙箱；同 UID 执行
- [ ] PAT scope 与安装器对接说明
- [ ] 构建事件默认与 Job 覆盖语义写入 DESIGN/操作说明（若尚未齐）

## 5. 验收清单

### 5.1 Gate 验收

- [ ] Claude Code、OpenCode、Reasonix、Codex 均可检测，且各有至少一条参考运行路径（健康或可观测失败）
- [ ] 手动、API、Cron、构建事件均可创建独立 AgentRun
- [ ] AgentRun 失败不影响已成功 BuildRun
- [ ] 合法 Skill ZIP 可上传；缺 `SKILL.md` 拒绝
- [ ] Skill 覆盖更新后下载为新包
- [ ] PAT 无效 → 401；scope 外 → 403；私钥/明文仅创建时回显一次
- [ ] 文档生成写入 draft，需人工发布

### 5.2 功能补充验收（对齐 PRD §10.7、§11.5）

- [ ] 智能体上下文仅为系统提示词 + 代码仓库（未偷偷注入需求/历史会话）
- [ ] 构建事件默认 `artifact_ready`；Job 可覆盖为 `distribution_finished`
- [ ] public/private Skill 隔离行为正确
- [ ] CLI 安装多源回退可观测（对齐工具链策略）
- [ ] 流水线中无同步 agent 阶段

### 5.3 契约 / 测试 / 质量门禁

- [ ] OpenAPI 已更新且投影已再生
- [ ] 版本化 migration + 三驱动合同测试
- [ ] 写操作与 PAT 创建/吊销有审计；token 明文不落日志
- [ ] 未引入远程 Runner、沙箱隔离承诺、Skill 版本历史

### 5.4 安全边界表述

- [ ] 产品/文档未声称 AI CLI 沙箱安全
- [ ] 未声称 PAT 或 password_cipher 替代 HTTPS/TLS
