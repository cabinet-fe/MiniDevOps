## 仪表盘

- 构建任务卡片
- 系统信息卡片
- 系统状态卡片

## 运维

运维面板不支持多用户, 仅超级管理员可用.

- 进程管理
- 开发环境管理
- 代理管理

## CI/CD

- 代码仓库
- 构建任务
- 服务器
- 凭证

## 项目管理

- 项目管理
- 需求管理
- 接口文档

## AI

- CLI管理

- 智能体
  - 智能体的定义
  - 智能体的上下文
  - 智能体触发器
  - 利用 CLI 进行操作
  - 智能体的交互记录(输入和输出)

- SKILLs管理
  - 分公共 SKILLs 和私有 SKILLs
  - 可以在外部使用 API 进行对接
  - 可以直接上传一个 SKILL 的压缩包
  - 增删改查
